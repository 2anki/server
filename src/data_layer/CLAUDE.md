# CLAUDE.md

## Kanel-generated types

- `public/` is auto-generated by Kanel — never edit those files manually
- After adding a migration, regenerate with `pnpm run kanel` (configured via `.kanelrc.js`)
- The server runs migrations automatically on startup

## Repositories

- Repositories are the only layer that accesses the database directly
- No business logic here — fetch and persist data, let callers decide what to do with it
